Forcer le plein écran sous Windows 10

Description

Windows 10 64 bits fait tourner correctement de nombreux jeux des années 2000, mais il est souvent impossible de passer correctement en mode "plein écran".
Les bordures de la fenêtre sont toujours présentes et cette fenêtre n'est pas centrée, ce qui empêche d'en voir la totalité.
J'ai donc écrit un petit programme qui cherche la fenêtre d'un jeu (voir la liste des jeux supportés juste en dessous), enlève la bordure de la fenêtre puis la passe en plein écran.

Jeux supportés

Pour l'instant, seuls deux jeux auxquels j'ai joués sont supportés par le programme. Envoyez-moi un mail si vous voulez rajouter un jeu.

  • Morrowind (et ses extensions)
  • Tomb Raider IV - The Last Revelation

Instructions d'utilisation

Il faut d'abord lancer Play_It_Fullscreen.exe puis le jeu. Dès que Play_It_Fullscreen.exe trouve la fenêtre du jeu, il va la passer en plein écran puis se fermer automatiquement.
Si vous obtenez l'erreur "Error : failed to set the window style", lancez Play_It_Fullscreen.exe en mode administrateur pour que le programme ait le droit d'envoyer des messages à la fenêtre du jeu.

Obtenir le programme

Ce programme compile sous Visual Studio 2010. Il a été testé sous Windows 10 64-bit Professional N build 10586.
Les sources sont disponibles sur GitHub.
